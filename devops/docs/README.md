# Deployment Documentation

This directory contains comprehensive deployment guides and documentation.

## Documents

- `DEPLOYMENT.md` - Complete deployment guide
- `QUICKSTART.md` - Quick start guide for getting started
- `GITHUB-OIDC-SETUP.md` - GitHub OIDC configuration guide
- `SECRETS-MANAGEMENT.md` - Secrets management best practices

## Getting Started

1. Start with `QUICKSTART.md` for a fast path to deployment
2. Refer to `DEPLOYMENT.md` for detailed instructions
3. Follow `GITHUB-OIDC-SETUP.md` to configure GitHub Actions authentication
4. Review `SECRETS-MANAGEMENT.md` for secure secret handling
