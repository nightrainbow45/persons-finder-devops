# DevOps Infrastructure

This directory contains all DevOps-related configurations for the Persons Finder application.

## Directory Structure

- `docker/` - Docker containerization configurations
- `helm/` - Helm Charts for Kubernetes deployment
- `terraform/` - Infrastructure as Code (Terraform modules and environments)
- `scripts/` - Deployment and automation scripts
- `ci/` - CI/CD pipeline configurations
- `docs/` - Deployment documentation and guides

1. Review the documentation in `docs/`
2. Configure your environment (AWS credentials, kubectl, helm)
3. Follow the deployment guide for your target environment
